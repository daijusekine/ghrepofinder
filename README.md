# GitHub Repository Finder
- You can search repositories on GitHub and bookmark them.
- Search result will be shown in order of stars.

# Requirement
Node.js
PostgreSQL
sequelize-cli

# How to set up
- Clone this repository.
- Set up PostgreSQL and configure `./config/config.json`
- `npm install`
- `sequelize db:migrate`
- `npm start`
- Access `http://localhost:3000/`
